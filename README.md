TWRP Device Tree Generator ToolHow to use this tool?First, you need to obtain an image from a functioning, bootable system on your device:boot.img for devices with A/B partitions.recovery.img for non-A/B partition devices.Fork this repository to your own GitHub account.Upload the recovery.img or boot.img to a location that provides a direct download link. It is highly recommended to upload the .img file directly to this repository, then click View Raw to obtain the direct link.Go to the Actions tab, select make twrp device, click Run workflow, and paste your direct link into the input box.Once filled out, click Run workflow to begin.Compilation ResultsYou can download the compiled files in the Releases section.
